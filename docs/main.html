Boron UI Library

A feature-rich, themeable Roblox UI library for building sleek in-game menus. Supports toggles, sliders, dropdowns, color pickers, key bindings, notifications, and more.

Roblox Lua 5.1 CoreGui Safe Rainbow Support

Setup

🚀 Getting Started

Load the library using loadstring or by requiring it directly. The library returns a Library object used to create windows and manage the entire UI.

Lua 
-- Load the library
local Library = loadstring(game:HttpGet('YOUR_LIBRARY_URL'))()

-- Access global state tables
local Toggles = getgenv().Toggles
local Options  = getgenv().Options
ℹ️

The library automatically parents itself to CoreGui and calls protectgui / syn.protect_gui if available.

Layout

🪟 Create Window

A Window is the top-level container. It holds all Tabs, which in turn hold Groupboxes and components.

Lua
local Window = Library:CreateWindow({
    Title        = 'My Script',
    Center       = true,          -- Center on screen
    AutoShow     = true,          -- Show immediately
    MenuFadeTime = 0.2,           -- Toggle fade duration
    TabPadding   = 0,             -- Padding between tabs
    Position     = UDim2.fromOffset(175, 50),
    Size         = UDim2.fromOffset(750, 550),
})
PropertyTypeRequiredDescription
TitlestringoptWindow title displayed at the top. Defaults to 'No title'.
CenterbooleanoptIf true, anchors the window at the center of the screen.
AutoShowbooleanoptAutomatically toggle (show) the menu on load.
MenuFadeTimenumberoptDuration in seconds for the show/hide fade animation. Default: 0.2.
PositionUDim2optStarting position. Default: UDim2.fromOffset(175, 50).
SizeUDim2optWindow size. Default: UDim2.fromOffset(750, 550).

Library:Toggle()

Toggles the visibility of the main menu with a fade animation. Bound by default to RightControl or RightShift. You can also bind a custom key via Library.ToggleKeybind.

💡

Set Library.ToggleKeybind = Options.MyKeypicker after creating a Key Picker to use it as the menu toggle.

Layout

📑 Tabs

Tabs are the top-level navigation inside a window. Each tab has a left column and a right column for groupboxes.

Lua
local Tab = Window:AddTab('Combat')

-- First tab is shown automatically
Tab:ShowTab()  -- manually show
Tab:HideTab()  -- manually hide
Layout

📦 Groupboxes

Groupboxes are containers inside a tab. They auto-resize as you add elements. Place them on the left or right side.

Lua
local LeftBox  = Tab:AddLeftGroupbox('Settings')
local RightBox = Tab:AddRightGroupbox('Info')

-- Or with full config:
local Box = Tab:AddGroupbox({
    Name = 'Settings',
    Side = 1,  -- 1 = left, 2 = right
})
Layout

🗂️ Tabboxes

Tabboxes are sub-tab containers that sit inside a column, letting you split content into nested pages.

Lua
local Tabbox = Tab:AddLeftTabbox()

local SubA = Tabbox:AddTab('Page A')
local SubB = Tabbox:AddTab('Page B')

-- Add elements to each sub-tab like a normal groupbox
SubA:AddToggle('MyToggle', { Text = 'Feature A' })
Layout

🔗 Dependency Box

A Dependency Box is a groupbox that shows or hides its contents based on the state of a Toggle. Useful for hiding advanced settings until a feature is enabled.

Lua
local Depbox = LeftBox:AddDependencyBox()

Depbox:AddSlider('MySlider', {
    Text     = 'Speed',
    Default  = 50,
    Min      = 0,
    Max      = 100,
    Rounding = 0,
})

-- Show Depbox only when 'SpeedToggle' is true
Depbox:SetupDependencies({
    { Toggles.SpeedToggle, true }
})
ArgTypeDescription
DependenciestableArray of { ToggleObject, expectedValue } pairs. All must match for the box to show.
Component

☑️ Toggle

A simple on/off boolean switch. Supports addons (Color Picker, Key Picker), tooltip, and a Risky flag that renders the label in red.

Lua
LeftBox:AddToggle('AutoFarm', {
    Text    = 'Auto Farm',
    Default = false,
    Tooltip = 'Automatically farms resources',
    Risky   = false,
    Callback = function(Value)
        print('Auto Farm:', Value)
    end
})

-- Read / write value
Toggles.AutoFarm:SetValue(true)
print(Toggles.AutoFarm.Value)

-- Listen for changes
Toggles.AutoFarm:OnChanged(function(v)
    print('Changed to:', v)
end)
PropertyTypeRequiredDescription
TextstringreqLabel displayed next to the checkbox.
DefaultbooleanoptInitial value. Defaults to false.
TooltipstringoptHover tooltip text.
RiskybooleanoptRenders the label in RiskColor (red) to warn users.
Callbackfunction(bool)optCalled whenever the value changes.
💡

Toggles are stored in the global Toggles table by their Idx string. Access them anywhere with Toggles.MyIdx.

Component

🎚️ Slider

A draggable horizontal slider for selecting a numeric value within a range.

Lua
LeftBox:AddSlider('WalkSpeed', {
    Text     = 'Walk Speed',
    Default  = 16,
    Min      = 0,
    Max      = 200,
    Rounding = 0,         -- 0 = integer, 1+ = decimal places
    Suffix   = ' stud/s',
    Compact  = false,      -- No label above slider
    HideMax  = false,      -- Hide the /Max part
    Tooltip  = 'Player walk speed',
    Callback = function(Value)
        game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = Value
    end
})

-- Set programmatically
Options.WalkSpeed:SetValue(50)
PropertyTypeRequiredDescription
TextstringreqLabel displayed above the slider.
DefaultnumberreqInitial value.
MinnumberreqMinimum value.
MaxnumberreqMaximum value.
RoundingnumberreqDecimal precision. 0 = integer.
SuffixstringoptText appended to the displayed value (e.g. ' ms').
CompactbooleanoptRemoves the label row; text shown inline on the slider.
HideMaxbooleanoptHides the /Max part of the display.
TooltipstringoptHover tooltip text.
Callbackfunction(number)optCalled every time the value changes while dragging.
Component

🔘 Button

A clickable button that fires a callback. Supports an optional second button inline and a double-click confirmation mode.

Lua
local Btn = LeftBox:AddButton({
    Text        = 'Teleport Home',
    Func        = function() print('teleporting!') end,
    DoubleClick = false,  -- Require double-click confirmation
    Tooltip     = 'Teleport to spawn',
})

-- Attach a second button (appears inline to the right)
Btn:AddButton({
    Text = 'Options',
    Func = function() print('options!') end
})
PropertyTypeRequiredDescription
TextstringreqButton label.
Funcfunction()reqCallback fired on click.
DoubleClickbooleanoptIf true, requires a second click within 0.5s to confirm.
TooltipstringoptHover tooltip text.
Component

📝 Input

A text input field with optional numeric validation, placeholder text, and a max length constraint.

Lua
LeftBox:AddInput('PlayerName', {
    Text        = 'Target Player',
    Default     = '',
    Placeholder = 'Enter name...',
    Numeric     = false,   -- Only allow numbers
    Finished    = false,   -- Fire only on Enter, not every keystroke
    MaxLength   = 64,
    Tooltip     = 'Name of target player',
    Callback    = function(Value)
        print('Input:', Value)
    end
})

-- Set programmatically
Options.PlayerName:SetValue('Roblox')
PropertyTypeRequiredDescription
TextstringreqLabel displayed above the text box.
DefaultstringoptInitial text content.
PlaceholderstringoptGreyed-out hint text when box is empty.
NumericbooleanoptOnly accepts numeric characters.
FinishedbooleanoptCallback fires only when Enter is pressed, not on every keystroke.
MaxLengthnumberoptTruncates input to this character count.
Callbackfunction(string)optFires on text change (or on Enter if Finished is true).
Component

📋 Dropdown

A scrollable dropdown list. Supports single and multi-select modes, as well as special types that auto-populate with players or teams.

Lua
-- Single select
LeftBox:AddDropdown('WeaponType', {
    Text      = 'Weapon Type',
    Values    = { 'Sword', 'Bow', 'Staff' },
    Default   = 'Sword',
    AllowNull = false,
    Callback  = function(Value) print(Value) end
})

-- Multi-select
LeftBox:AddDropdown('Features', {
    Text    = 'Active Features',
    Values  = { 'ESP', 'Aimbot', 'Speed' },
    Default = { 'ESP' },
    Multi   = true,
    Callback = function(Value) -- Value is a dict {name=true}
        for k, _ in next, Value do print(k) end
    end
})

-- Auto player list
LeftBox:AddDropdown('Target', {
    Text        = 'Target Player',
    SpecialType = 'Player',  -- auto-fills + updates
})

-- Update values at runtime
Options.WeaponType:SetValue('Bow')
Options.WeaponType:SetValues({ 'Axe', 'Spear' })
PropertyTypeRequiredDescription
ValuestablereqArray of string options.
Defaultstring / tableoptDefault selected value(s). Required unless AllowNull = true.
MultibooleanoptAllow selecting multiple values at once.
AllowNullbooleanoptAllow no selection (value will be nil).
SpecialType'Player' / 'Team'optAuto-populate with the current player/team list. Updates live.
TextstringoptLabel above the dropdown. If omitted, no label row is shown.
CallbackfunctionoptFires when selection changes. Value is a string or dict (multi).
Component

🏷️ Label

A static text element. Can optionally wrap long text across multiple lines.

Lua
local Lbl = LeftBox:AddLabel('Version 1.0.0')
local LblWrap = LeftBox:AddLabel('This is a longer description that wraps.', true)

-- Update the text dynamically
Lbl:SetText('Version 1.0.1')
ℹ️

A non-wrapping label also supports addons (Color Picker, Key Picker) via the BaseAddons metatable — useful for labelled side controls.

🎨 Color Picker

Attaches a compact color swatch to a Toggle or Label. Clicking it opens an HSV picker with Hex and RGB text inputs. Right-clicking opens a context menu to copy/paste colors.

Lua
LeftBox:AddToggle('ESPEnabled', { Text = 'ESP', Default = true })
    :AddColorPicker('ESPColor', {
        Title        = 'ESP Color',
        Default      = Color3.fromRGB(255, 100, 100),
        Transparency = 0,  -- enable transparency slider when set
        Callback     = function(Color)
            print('Color:', Color)
        end
    })

-- Programmatic access
Options.ESPColor:SetValueRGB(Color3.fromRGB(0, 255, 0))
Options.ESPColor:SetValue({ Hue, Sat, Vib }, Transparency)
print(Options.ESPColor.Value)  -- Color3

Options.ESPColor:OnChanged(function(Color)
    print('New color:', Color)
end)
PropertyTypeRequiredDescription
DefaultColor3reqInitial color value.
TitlestringoptTitle shown inside the picker popup.
TransparencynumberoptSet to any value to show a transparency slider (0 = fully opaque).
Callbackfunction(Color3)optFires whenever the color changes.
Addon

⌨️ Key Picker

Attaches a keybind button to a Toggle or Label. Left-click to rebind; right-click to change mode. Appears in the Keybind HUD automatically.

Lua
Toggles.AutoFarm:AddKeyPicker('AutoFarmKey', {
    Default          = 'F',
    Text             = 'Auto Farm',
    Mode             = 'Toggle',  -- 'Always' | 'Toggle' | 'Hold'
    Modes            = { 'Toggle', 'Hold' },  -- visible modes
    SyncToggleState  = true,   -- keep parent toggle in sync
    NoUI             = false,  -- hide from Keybind HUD
    Callback         = function(State)
        print('Keybind state:', State)
    end,
    ChangedCallback = function(NewKey)
        print('Rebound to:', NewKey)
    end
})

-- Read state
if Options.AutoFarmKey:GetState() then
    -- key is active
end

-- Set key + mode programmatically
Options.AutoFarmKey:SetValue({ 'G', 'Hold' })
PropertyTypeRequiredDescription
DefaultstringreqInitial key name (e.g. 'F', 'MB1', 'None').
TextstringoptLabel shown in the Keybind HUD.
ModestringoptInitial mode: 'Always', 'Toggle', or 'Hold'. Default: 'Toggle'.
ModestableoptWhich mode buttons appear in the right-click menu. Default: all three.
SyncToggleStatebooleanoptAutomatically sync the parent Toggle's value with this keybind.
NoUIbooleanoptIf true, this keybind will not appear in the Keybind HUD.
Callbackfunction(bool)optFires when toggled or pressed.
ChangedCallbackfunction(KeyCode)optFires when the user rebinds the key.

Mouse Buttons

Use 'MB1' for left mouse button and 'MB2' for right mouse button as the key value.

Customization

🎨 Theming

All colors are stored on the Library table and can be changed at any time. Call Library:UpdateColorsUsingRegistry() after making changes to apply them to all existing elements.

Lua
-- Default color values
Library.FontColor       = Color3.fromRGB(255, 255, 255)
Library.MainColor       = Color3.fromRGB(28,  28,  28)
Library.BackgroundColor  = Color3.fromRGB(40,  40,  40)
Library.BackgroundColor2 = Color3.fromRGB(30,  30,  30)
Library.AccentColor      = Color3.fromRGB(255, 0,   0)  -- highlights, active states
Library.OutlineColor     = Color3.fromRGB(59,  59,  59)
Library.GroupboxColor    = Color3.fromRGB(35,  35,  35)
Library.TabActiveColor   = Color3.fromRGB(50,  50,  50)
Library.RiskColor        = Color3.fromRGB(255, 50,  50)

-- Change accent to blue
Library.AccentColor     = Color3.fromRGB(0, 120, 255)
Library.AccentColorDark = Library:GetDarkerColor(Library.AccentColor)
Library:UpdateColorsUsingRegistry()  -- apply everywhere

-- Rainbow accent (auto-updates every frame)
-- Library.CurrentRainbowColor is updated automatically
-- hook it via RunService if you want rainbow accent:
game:GetService('RunService').RenderStepped:Connect(function()
    Library.AccentColor     = Library.CurrentRainbowColor
    Library.AccentColorDark = Library:GetDarkerColor(Library.AccentColor)
    Library:UpdateColorsUsingRegistry()
end)

💧 Watermark

A small draggable label shown in the top-left corner of the screen, useful for displaying script name or status.

Lua
Library:SetWatermark('MyScript v1.2 | ' .. game.Players.LocalPlayer.Name)
Library:SetWatermarkVisibility(true)   -- show
Library:SetWatermarkVisibility(false)  -- hide

🔔 Notifications

Toast-style animated notifications that appear in the top-left and auto-dismiss after a timer.

Lua
Library:Notify('Settings saved!')          -- default 5s
Library:Notify('Quick message', 2)          -- 2s
Library:Notify('Error: something went wrong', 8)  -- 8s
ParameterTypeDescription
TextstringThe message to display.
TimenumberDuration in seconds before the notification fades out. Default: 5.
Global UI

🎹 Keybind HUD

A draggable HUD panel that automatically lists all registered Key Pickers and their current state. Shown/hidden separately from the main window.

Lua
-- Show the keybind frame
Library.KeybindFrame.Visible = true

-- It auto-resizes as keybinds are added/removed
Global State

🗃️ Toggles & Options

All UI elements are registered in two global tables: Toggles for toggle components, and Options for everything else (sliders, dropdowns, inputs, color pickers, key pickers).

Lua
-- Toggle
Toggles.AutoFarm.Value           -- boolean
Toggles.AutoFarm:SetValue(true)

-- Slider
Options.WalkSpeed.Value           -- number
Options.WalkSpeed:SetValue(100)

-- Dropdown
Options.WeaponType.Value          -- string (single) or {name=true} (multi)
Options.WeaponType:SetValue('Bow')

-- Color picker
Options.ESPColor.Value            -- Color3
Options.ESPColor.Transparency    -- number 0–1

-- Key picker
Options.AutoFarmKey.Value         -- string key name
Options.AutoFarmKey:GetState()    -- boolean: currently active?
